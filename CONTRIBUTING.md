# Contributing to Awesome MCP Servers for Sales Automation

First off, thank you for considering contributing to this list! It's people like you that make it a valuable resource for the community.

Please take a moment to review these guidelines before submitting a pull request.

## Guidelines

1. **Only Real, Functional MCP Servers** - We only accept servers that actually work, have a public repository or service endpoint, and are directly useful for sales automation, outreach, lead generation, CRM, databases, or scheduling workflows.
2. **Formatting** - Please keep the formatting consistent:
   - Format: `[Tool Name](github-link) - One line description. ⭐ if notable`
   - Description should start with a capital letter and end with a period.
   - Limit description to one concise line focusing on what the tool does.
3. **Alphabetical Order** - Within each category, please insert new tools alphabetically to keep the list clean and easy to scan.
4. **Awesome-List Quality** - Ensure your description is objective, clear, and avoids marketing speak or hype.
5. **Stars (⭐)** - Stars are reserved for official implementations by the core product teams (e.g., Anthropic, HubSpot, Salesforce, Microsoft, Cal.com) or exceptionally popular/widely used community projects (500+ GitHub stars).

## How to Contribute

1. Fork the repository.
2. Create a new branch with a descriptive name (e.g., `add-mytool-mcp`).
3. Make your changes in `README.md`.
4. Commit your changes with a clear message (e.g., `Add mytool-mcp to Email Automation`).
5. Push your branch and submit a pull request against the main branch.
6. Provide a quick summary in the pull request explaining what the tool is and why it belongs on this list.

Thank you for your contribution!
